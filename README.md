# Chat Screenshot Analyzer — результаты прогона

Витрина результатов тестового задания: каждый присланный скриншот переписки прогнан
через n8n-воркфлоу (Gemini vision + structured output). Для каждого скрина показан
**сырой вывод воркфлоу** в формате ТЗ (по умолчанию) и наглядный **разбор** (по тумблеру).

Акцент задания — корректное определение **сторон переписки** (по выравниванию пузыря,
не по цвету) и **платформы** (по UI-сигнатурам: Telegram / WhatsApp / Instagram / Tinder /
Twinby / Неизвестно), с делением на мессенджер / сайт знакомств.

**Страница:** `index.html` (открывается локально или через GitHub Pages).
Сборка: `node build.mjs` — карточка задаётся своим сырым выводом, наглядный вид парсится из него.
